A Translation Dashboard Block works with Translation Dashboard plugin

== Description ==

A Translation Dashboard Block works with Translation Dashboard plugin. This block can be placed with the full site editor in any template or page. You get an options to select any language which has posts or pages assigned in the website by Translation Dashboard plugin.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/trank-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
